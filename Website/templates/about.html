SNPip

Single nucleotide polymorphisms (SNPs) are a type of polymorphism that involves a variation at a single nucleotide base-pair. SNPs are extensivley studied to see how they relate to disease, treatment response, and other characteristics in humans and other organisms.

Inputs


The SNPip web application will allow users to retrieve SNP information given either a genomic coordinate (chromosome, start and end), SNP name (rs value), or gene name (or any aliases associated to it) from specific populations in the 1000 genome project.



Outputs


The application will return the following information for each SNP:

  • Name (rs value)
  • Genomic position
  • Genotype frequencies
  • Allele frequency

If multiple SNPs are returned, the user will be able to select the population(s) and summary statistics of interest, and the application will calculate them and plot their distribution in sliding windows along the region. The user will also be able to download a text file with the values of summary statistics. At least three summary statistics should be reported, for instance:

  • Genetic diversity
  • Haplotype diversity (e.g. homozygosity)
  • Test against neutrality (e.g. Tajimaâ€™s D)
  • Population genetic variation (FST value) for each pair of populations
